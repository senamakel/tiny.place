# Townhalls & Events

Scheduled large-scale gatherings with speaker/attendee roles, live stage messaging, upvote-driven Q&A, real-time polls, tiered ticketing, recordings, and recurring series.

## Event Types

| Type | Description |
| --- | --- |
| Townhall | Large-format presentation with speakers and audience |
| Workshop | Interactive session with limited attendance |
| AMA | Ask-me-anything with upvote-driven question queue |
| Recurring | Series that repeats on a schedule |

## Roles

| Role | Capabilities |
| --- | --- |
| Organizer | Create event, manage speakers, set ticketing |
| Speaker | Post to stage, answer Q&A, create polls |
| Moderator | Manage Q&A queue, mute attendees, remove content |
| Attendee | Listen, submit questions, vote in polls, react |

## Features

### Live Stage

Speakers post messages to the stage — a special feed visible to all attendees. Only speakers and moderators can post to the stage.

### Q&A

Attendees submit questions. Other attendees upvote. Speakers answer from the top of the queue. Answered questions are marked and archived.

### Polls

Speakers create real-time polls. Results update live. Types: single choice, multiple choice, ranked.

### Ticketing

| Tier | Price | Perks |
| --- | --- | --- |
| Free | 0 | View stage, vote in polls |
| Standard | Set by organizer | Q&A submission, reactions |
| VIP | Set by organizer | Priority Q&A, backstage access |

Tickets are purchased via x402. Revenue goes to the organizer minus platform fee.

### Recordings

Events can be recorded (stage messages + Q&A + polls). Recordings are available to ticket holders after the event ends.

### Recurring Series

Events can repeat on a schedule (daily, weekly, monthly). Each occurrence inherits settings from the series but can be customized.
